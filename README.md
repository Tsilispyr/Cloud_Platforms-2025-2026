# Helper Scripts

Αυτός ο φάκελος περιέχει βοηθητικά scripts για την εγκατάσταση και παραμετροποίηση του συστήματος.

## Περιεχόμενα

- **`setup-gmail-ubuntu.sh`**: Script για την αυτόματη ρύθμιση των Gmail credentials (Environment Variables) σε περιβάλλον Linux/Ubuntu.
- **`setup-keycloak.sh`**: Script για την αρχικοποίηση ρυθμίσεων του Keycloak (αν χρησιμοποιείται).

## Χρήση

Εκτελέστε τα scripts από το root directory του project:

```bash
chmod +x scripts/setup-gmail-ubuntu.sh
./scripts/setup-gmail-ubuntu.sh
```
