# Frontend Documentation

## Overview
Το Frontend είναι η διεπαφή χρήστη για το DevOps Pets System.

## Access
Μετά το deployment, το frontend είναι διαθέσιμο στο:
- **URL**: http://localhost:8083

## Features
- Εγγραφή και Σύνδεση χρηστών.
- Προβολή λίστας ζώων.
- Διαχείριση προφίλ χρήστη.
- Επικοινωνία με το Backend API στο port 8080.
